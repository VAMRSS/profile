Export / JSON Backup

登録済みの作品・シリーズ・各種候補データをJSON形式で書き出します。定期バックアップや別端末への移行に使用します。

Import / JSON Restore

WSV形式のJSONバックアップを読み込みます。読み込み後、現在のブラウザ内データはバックアップ内容で置き換えられます。

CSV Import

CSVから初期データを取り込みます。先にSERIES CSV、次にWORKS CSVを読み込んでください。タイトルと説明文は主項目とEN項目で管理し、サイズはsizeH,sizeW,sizeDのH/W/D順で取り込みます。
